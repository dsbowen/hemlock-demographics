Hemlock demographics

Hemlock-demographics is a hemlock extension for collecting demographic information.

Installation

$ hlk install hemlock-demographics

$ pip install hemlock-demographics

Quickstart

In survey.py:

from hemlock import Branch, Page, Label, route
from hemlock_demographics import comprehensive_demographics

@route('/survey')
def start():
    return Branch(
        comprehensive_demographics(page=True),
        Page(
            Label('The end.'), 
            terminal=True
        )
    )

Create a file app.py:

import eventlet
eventlet.monkey_patch()

from hemlock import Branch, Page, Label, create_app, route
from hemlock_demographics import comprehensive_demographics

@route('/survey')
def start():
    return Branch(
        comprehensive_demographics(page=True),
        Page(
            Label('The end.'), 
            terminal=True
        )
    )

app = create_app()

if __name__ == '__main__':
    from hemlock.app import socketio
    socketio.run(app, debug=True)

Running your app

$ hlk serve

$ python3 app.py

Go to http://localhost:5000/ in your browser.
